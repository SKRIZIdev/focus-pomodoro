# 🍅 FocusFlow — Pomodoro Timer

A clean Pomodoro timer with an animated ring, focus/short-break/long-break modes, an inline task list, and a stats page that counts your focus sessions and minutes over the last 7 days.

**Live demo:** https://skrizidev.github.io/focus-pomodoro/

## Pages
- **Timer** — SVG ring countdown, mode switch, task list
- **Stats** — sessions today / all-time, minutes focused, 7-day chart
- **About** — the Pomodoro Technique

## Tech
Vanilla HTML / CSS / JavaScript. Sessions and tasks in `localStorage`.
